# 数模备赛 — 2026 CUMCM A题求解

## 赛题

2026年全国大学生数学建模竞赛 A题：**无人机投放烟幕干扰弹策略优化**

核心场景：无人机 FY1 携带烟幕干扰弹，在导弹 M1 飞向真目标途中释放烟幕，干扰导弹视线，最大化有效遮蔽时长。

## 核心方法

### 问题一
- 单位球面投影法：将三维遮蔽判定转化为单位球面上球冠覆盖问题
- 凸集分离定理保证只需验证目标圆柱边界采样点
- 定步长扫描 + 二分查找，精度达 $1.8\times10^{-15}\,\mathrm{s}$

### 问题二
- **几何预测算法**：沿 M1→目标视线采样候选烟幕中心，反解 FY1 参数生成优质起点
- **两阶段 SLSQP**：sigmoid 平滑近似提供梯度 → 离散精确目标精收敛
- 与 DE、PSO、变步长坐标搜索等方法进行了系统对比

### 问题三
- **差分进化 (DE)**：几何播种 + 自适应缩放因子 + 早期停止，5/5 收敛至 5.85s
- **系统扰动精搜**：网格扰动 + 局域精调，进一步提升至 6.16s
